# OpenCL basic initialization, get some device info, run simple kernel 

Compiled with gcc under LinuxMint 19

## Compile and Run

```sh
$ git clone https://github.com/tapin13/openCL-helloWorld.git
$ cd openCL-helloWorld
$ make
$ ./openCL-helloWorld
```
